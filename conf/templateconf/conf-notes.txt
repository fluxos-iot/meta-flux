
### Shell environment set up for builds. ###

You can now run 'MACHINE=<machine> DISTRO=<distro> bitbake <target>'

Common machines are:
    qemux86-64 (default)

Common distros are:
    flux (default)

Common targets are:
    flux-image-essential

You can also run generated qemu images with a command like 'runqemu qemux86-64'
